COMO EXECUTAR O PROTÓTIPO:
1. Instalar Python 3.10+ e PostgreSQL
2. Instalar ollama (https://ollama.com) e carregar um modelo:
   ollama pull llama3
3. Instalar dependências Python:
   pip install psycopg2-binary requests
4. Criar e popular o banco:
   createdb copa_do_mundo
   psql copa_do_mundo < sql/05.DDL.sql
   psql copa_do_mundo < sql/06.DML.sql
5. Executar o protótipo:
   cd prototipo
   python main.py
6. Inserir os parâmetros de conexão quando solicitado
